# ZynkChat App 📱✨

A beautiful, premium real-time chat application built with **Flutter** and **Riverpod**. Designed with a focus on rich aesthetics, smooth animations, and seamless real-time communication.

## 🎨 Aesthetics & Design
- **Modern Dark Theme**: Deep slate and obsidian palette with indigo accents.
- **Micro-animations**: Subtle transitions for a premium feel.
- **Premium Typography**: Uses the **Outfit** font for a clean, professional look.
- **Responsive Layout**: Optimized for all mobile devices.

## 🚀 Features
- [x] **Real-time Messaging**: Powered by Socket.io.
- [x] **Authentication**: Secure JWT-based Login/Register.
- [x] **Persistence**: Remembers your session across app restarts.
- [x] **User Discovery**: Search and find friends instantly.
- [x] **Typing Indicators**: See when someone is replying.
- [x] **Offline Connectivity**: Smart handling of connection states.

## 🏗️ Architecture
- **State Management**: flutter_riverpod (Reactive & Testable).
- **Networking**: Dio (with Interceptors for Auth).
- **Real-time**: socket_io_client.
- **Security**: flutter_secure_storage.

## 🛠️ Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Node.js Backend](https://github.com/muhammedshamil8/zynkchat-api) (Running)

### Installation
1.  **Clone the app**:
    ```bash
    git clone https://github.com/muhammedshamil8/zynkchat_app.git
    cd zynkchat_app
    ```
2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
3.  **Run the app**:
    ```bash
    flutter run
    ```
